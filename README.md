- to create a virtual env :
    ```
    python -m venv myvenv
    ```

- to activate it : 
    ```
    source myvenv/bin/activate
    ```

- to installl packages :
    ```
    pip install cowsay
    ```

- to deactivate it :
    ```
    deactivate
    ```

-  /!\  to delete it completely  /!\ :
    ```
    rm -rf myvenv
    ```
